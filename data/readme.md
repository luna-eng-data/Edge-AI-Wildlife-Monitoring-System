# Sources des Données

Les images utilisées pour l'entraînement ne sont pas hébergées sur ce repository pour des raisons de taille.

## Origine des datasets
Le dataset final a été agrégé à partir de deux sources Kaggle :

1. **African Wildlife** (Animaux)
   - Classes : Buffalo, Elephant, Rhino, Zebra
   - Source : Kaggle

2. **Human Detection Dataset** (Humains)
   - Classes : Personnes en milieu extérieur
   - Source : Kaggle

## Structure locale
Pour reproduire l'entraînement, les données doivent être organisées comme suit :
dataset_classification/
├── animaux/
└── humains/
